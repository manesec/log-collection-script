# log-collect-script

simple soc log collection with REST API.

## Config

In `/etc/log-collection`


## Location

`/usr/local/log-collection`

## example 

`curl -X PUT -H "Content-Type: application/json" -d '{"type":"file_name","data":"<base64_json_data>"}' http://127.0.0.1:8000`
